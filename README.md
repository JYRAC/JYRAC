# 日本若者防災復興協会 (JYRAC) 公式サイト

日本若者防災復興協会（特定非営利活動法人 設立登記申請中）の公式ウェブサイトです。
若者が主体となる防災・減災・災害復興活動の法的・財政的基盤を支え、地域や世代を超えた協働を促進するための情報発信プラットフォームとして開発されています。

従来のHTML構成から **Astroフレームワーク** へ移行し、軽量かつ高速なモダンウェブサイトへと刷新されました。

## 🚀 実装されている機能・特徴

- **高速な静的レンダリング (Astro):** 不要なJavaScriptを排除し、最高水準のページ速度とSEO（検索エンジン最適化）を実現。
- **スタイリッシュなスプラッシュ画面:** サイトを開いた際に、ロゴとシームレスなゲージが走る美しいローディング演出。
- **動的なお知らせ（アコーディオン）:** GSAP（GreenSock）を使用した、タイムライン制御による滑らかな開閉アニメーション。
- **直感的なタブ切り替えダッシュボード:** メインパネルを切り替える洗練されたUIインターフェース。
- **CMS連携対応設計:** 将来的なヘッドレスCMS（microCMSなど）の導入を見据え、フロントマターでデータを一元管理できる柔軟なコード構造。
